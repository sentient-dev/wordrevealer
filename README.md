# WordRevealer

WordRevealer is a lightweight web app that gradually reveals letters from a hidden word or phrase.

## Quick start

1. Open `index.html` directly in your browser.
2. Enter any word or phrase.
3. Click **Set Secret**, then use **Reveal Next** to uncover it letter by letter.

## Files

- `index.html`: app structure
- `styles.css`: visual design and responsive layout
- `script.js`: reveal interaction logic
